# VideoAnalysisEngine
A video analysis engine with similarity frame retrieval,keyframe faces recognition,keyframe objects detection and etc.

# Requirments
- ubuntu 18.04 (recommended)
- python 3
- go 1.10
- gcc 7
- g++ 7
- postgresql 10

# Dependencies
- boost 1.62.0.1
- opencv 3.1
- vlfeat 0.9.20
- yael  https://github.com/jackculpepper/yael
- libyaml-cpp 0.5.2
- pqxx 4.0.1
- protobuf 3.5.1
- libgrpc++ 
- liblas-dev
